# React

这是一个用`webpack`构建的一个具有react环境的todolist项目，其中包括了：

- react组件的简单写法
- react中简单的jsx语法
- 容器组件中的this指向问题
- jsx语法细节补充
- 组件拆分(容器组件和UI组件)
- propTypes + defaultProps
- react的生命周期函数
- react的简单动画
- redux
- actionTypes + actionCreators
- redux-saga + redux-thunk

#### 如果你对react没有一点了解的话，建议先去学一下react的基本语法和思想
